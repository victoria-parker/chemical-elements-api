Chemical Elements API

API Usage

Get all elements

To receive an array with all the elements of the table this is the endpoint:

                'https://chemical-elements-api.herokuapp.com/elements'

Get elements by atomic number

In case you want to receive just one element in particular, you can get it by its atomic number.This would be the endopoint for Hydrogen for example:

                'https://chemical-elements-api.herokuapp.com/elements/1'

Here is how each element object looks like, using the Hydrogen as example:
